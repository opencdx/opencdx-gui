OpenCDx System Overview

OpenCDx provides an open framework, specification, and set of developer tools – including client libraries, Software Development Kits (SDKs), and Application Programming Interfaces (APIs) – to accelerate development and adoption of interoperable, standards-based CDx applications that can be leveraged for a wide variety of public and private sector use cases.

Key Services

  • OpenCDx-Admin: This is the admin portal for OpenCDx. Developed using Spring Admin Server, it provides backend monitoring capabilities specifically for the OpenCDx environment.
  • OpenCDx-ANF: ANF aims to enhance data structure within HL7 v2 messages for more effective data analysis. It involves categorization of data into logical segments based on their interrelationships and usage frequency. This allows for better understanding, data manipulation, and ease of analysis.
  • OpenCDx-Audit: This module records key actions performed by individual users, systems, or services on OpenCDx. It creates audit records for every service invocation and user interaction on the platform.

Benefits

  • Connected Diagnostics: Facilitating standards-based diagnostic data capture, harmonization, and transmission between disparate systems within the existing healthcare ecosystem.
  • Community Focused: Remote and connected diagnostics adoption throughout the healthcare industry in an effort to support public health and national security interests.
